# Changelog

## 0.2.0 - 2026-02-28
- Added persistent app version metadata endpoint (`/app-meta`).
- Added bottom-left version badge support in web pages with latest change notes.
- Added Pokemon discovery product table with product/store/price columns.
- Added additional Pokemon discovery sources (Amazon, Pokemon Center).
- Added concert ticket search improvements: event ID support, section filter, max price filter, and 50-mile radius support.
- Hardened ticket provider link quality with validated/fallback URLs.

## 0.1.0 - 2026-02-28
- Initial release with Pokemon stock checking, Discord alerts, and web UI.
